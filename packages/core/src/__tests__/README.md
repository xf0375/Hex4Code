# About Full Test Suite

## Why No Full Test Suite

Hex4Code's core technology involves deep domain knowledge in the following areas:

- **DualTrit Compression** — Context compression algorithm based on ternary encoding structures; correctness verification requires understanding of coding theory
- **TC Propagation (Tropical Calculus)** — Trust propagation mechanism over tropical semirings, involving discrete mathematics
- **Multi-Model Intelligent Routing** — Task type classification and model selection strategies, requiring real provider response validation
- **Semantic Cache** — Semantic similarity computation and caching strategies, involving vector matching

Verifying the correctness of these technologies requires strong domain expertise, making it inherently challenging to write meaningful full-coverage tests. Adding large amounts of shallow tests (e.g., "state tests" that mock all external dependencies) could mislead users into thinking the core logic is covered by tests.
